Delegated Operations & Job Orchestration

Welcome to the DOJO — a self-hosted AI agent orchestration platform for macOS.

This installer will:

• Install Homebrew and Node.js (if needed)
• Set up the DOJO platform and watchdog service
• Install the DOJO menu bar app
• Start the server and open the setup wizard

Requires macOS 13 (Ventura) or later and 8GB+ RAM.
